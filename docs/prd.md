# [PRD] 조달 데이터 기반 전략 엔진: ProcureVision (가칭)

**버전:** 1.1 (shadcn/ui 기반 UI 전략 반영)
**작성일:** 2025. 12. 18
**상태:** 초안 (Draft)

---

## 1. 프로젝트 개요 (Project Overview)
본 프로젝트는 공공조달(나라장터) 시장에서 발생하는 파편화된 데이터를 수집·분석하여, '제품 규격(Spec)'과 '시계열 단가(Price)'를 결합한 인사이트를 제조/건설 실무자에게 제공하는 전략 대시보드 구축을 목표로 합니다.

* **비전:** "엑셀 노가다 없는 데이터 기반 조달 전략의 대중화"
* **핵심 가치:** 정보 불균형 해소, 입찰 성공률 제고, 실시간 경쟁 모니터링
* **해결하려는 핵심 문제 (Critical Gap):** 제품 사양(Spec)과 결합된 시계열 단가 분석 도구의 부재 해결

---

## 2. 타겟 고객 및 사용자 페르소나 (Target Audience)
1.  **전략기획 (김민수 팀장):** 경쟁사 가격 추이를 분석해 우리 제품의 단가 방어 전략 수립.
2.  **영업지원 (이지혜 과장):** 카테고리별 시장 점유율 랭킹을 시각화하여 경영진 보고용 자료 추출.
3.  **신규사업 (박준호 대표):** 신규 진입 품목의 스펙별 시장가와 마진 구조 시뮬레이션.
4.  **자재구매 (최성훈 차장):** 수천 개의 조달 제품 중 현장 사양에 맞는 최적의 단가 기준 수립.
5.  **투자심사 (정유리 심사역):** 비상장 제조사의 실시간 수주 모멘텀과 제품별 시장 지배력 검증.

---

## 3. 핵심 사용자 스토리 (User Stories)
* **분석:** "전략기획 팀장으로서, 특정 업체가 지난 1년간 50W LED 보안등의 단가를 어떻게 변경해 왔는지 라인 그래프로 확인하고 싶다."
* **필터링:** "영업지원 담당자로서, '직접생산' 업체들만 필터링하여 실제 제조사들의 시장 점유율만 따로 보고 싶다."
* **AI 인사이트:** "신규 사업팀으로서, 수십 페이지의 규격서(PDF)를 일일이 읽지 않고 AI가 요약해준 핵심 스펙 비교표를 보고 싶다."

---

## 4. 기능적 요구사항 (Functional Requirements)

### 4.1. 사용자 인터페이스 및 경험 (Next.js & shadcn/ui)
* **전문적 대시보드 레이아웃:** `shadcn/ui` 라이브러리를 활용하여 일관성 있고 고도화된 기업용 UI 제공.
* **인터랙티브 데이터 시각화:** 카드(Cards), 탭(Tabs), 데이터 테이블(Data Tables) 컴포넌트를 사용하여 복잡한 조달 데이터를 직관적으로 표시.
* **사양 기반 검색 필터:** 제품의 핵심 사양(W, lm, 인증 등)을 다중 선택하여 시장 상황을 좁혀볼 수 있는 필터링 기능.

### 4.2. 데이터 관리 및 인증 (Supabase)
* **기업용 보안 인증:** Supabase Auth를 통한 이메일 기반 로그인 및 보안 강화.
* **조달 데이터 허브:** 나라장터 API로 수집된 로우 데이터를 사양별로 정형화하여 저장 (PostgreSQL).
* **파일 스토리지:** 각 공고의 규격서(PDF, Word) 파일을 안전하게 저장하고 관리 (Supabase Storage).

### 4.3. AI 지능형 분석 (OpenAI)
* **자동 규격 파싱:** OpenAI(GPT-4o)를 활용하여 비정형 PDF 규격서에서 핵심 제품 사양을 자동으로 추출.
* **전략 요약 알림:** 대시보드 데이터를 분석하여 "A사의 점유율 급증"이나 "평균 단가 하락" 등의 인사이트를 자연어로 제공.

---

## 5. 비기능적 요구사항 (Non-functional Requirements)
* **UI 일관성:** Tailwind CSS와 `shadcn/ui`의 테마 시스템을 활용하여 모던하고 신뢰감 있는 디자인 유지.
* **반응형 최적화:** 맥북(MacBook) 및 태블릿 등 다양한 업무 환경에서 최적의 해상도 지원.
* **데이터 보안:** Supabase RLS(Row Level Security)를 적용하여 기업 간 데이터가 섞이지 않도록 철저히 격리.

---

## 6. 테크 스택 (Tech Stack)
* **Frontend:** Next.js (App Router), Tailwind CSS, shadcn/ui
* **Backend/DB/Auth:** Supabase
* **AI Engine:** OpenAI API (GPT-4o)

---

## 7. UI/UX 컨셉 설계 (shadcn/ui 구성)
* **Main Shell:** `Sidebar` 형태의 내비게이션과 상단 `Search Bar` 구성.
* **Summary Cards:** 당월 수주 총액, 전월 대비 변동률, 신규 진입 업체 수 등을 요약하여 상단 배치.
* **Analysis Section:** 중앙에 시계열 단가 추이 그래프(`Tabs` 활용)와 하단에 상세 제품 리스트(`Data Table`) 배치.
* **AI Panel:** AI가 분석한 '주간 핵심 전략 브리핑' 영역 배치.

---

## 8. 로드맵 (Roadmap)
* **Phase 1:** 데이터 파이프라인 구축 및 핵심 차트(점유율/단가) 구현.
* **Phase 2:** OpenAI 연동 규격서 자동 분석 및 사양 기반 필터링 고도화.
* **Phase 3:** 사용자 맞춤형 알림 설정 및 지능형 입찰가 추천 엔진 탑재.
